# show-project-setup

Generate a self-contained HTML dashboard showing the complete Claude Code environment setup.

## Skills

- `/show-project-setup` — Generate and open the dashboard

## What it shows

- Git info (remote, branch, project name)
- CLAUDE.md summaries (project + global)
- Rules, skills, plugins, MCP servers with YAML frontmatter
- Global settings and permissions
- Dev tools and environment detection
- Plugin hooks with badges and warnings
- Project type detection with suggested tools
